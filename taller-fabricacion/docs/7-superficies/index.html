7 - Superficies

Modelado a partir de superficies.

En este tutorial vamos a crear un modelo 3D a partir del módulo de superficies de Fusion 360. Lo que nos permitirá la creación de modelos 3D altamente personalizables y flexibles.

Para poder comenzar generamos un sketch aleatorio en el plano xy, para poder luego proyectar la forma deseada. Gracias a las cualidades de Fusion 360, no es necesaria la creación de un sketch totalmente preciso, ya que luego podremos generar cambios en el sketch en caso de ser necesario.

Para continuar, creamos un plano de construcción a 100 mm en el eje z, que nos permitira la creación de un segundo sketch

En este plano creamos otro sketch, que nos permitira la creación de una superficie sinuosa y muy perzonalizable.

Para unir ambos skecth, y finalmente crear una superficie, ocupamos el comando loft.

Para seguir personalizando nuestra superficie, crearemos otro sketch, pero esta vez en el plano lateral, que nos permitirá la creación de un hueco dentro de nuestra superficie.

Luego de crear el skecth, generamos una extrusión en ambos lados, sobrepasando los límites de nuestra superficie.

Para mejorar el resultado y quedarnos solamente con lo deseado, hacemos una división, que nos permitirá selecionar parte por parte.

Para selecionar aquellos elementos desados, hacemos un recorte seleccionando la zona central de nuestro modelo, eliminando las partes restantes.

Para poder unir estas superficies resultantes, vamos a la herramienta “coser”, que unirá estas superficies generando un sólido.

Luego, para una mejor terminación, realizamos un fillet a nuestro sólido.

Como mencionamos anteriormente, estamos trabajando con un sólido, entonces para poder generar un vacío en nuestro modelo ocuparemos la herramienta de vaciado, dando un espesor de 3mm, dandole forma final a nuestro recipiente.
