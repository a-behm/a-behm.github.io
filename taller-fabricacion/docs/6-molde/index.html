6 - Molde

Creación de molde

En este tutorial vamos a crear un molde apartir de un objeto 3D, para poder generar copias por vaciado de resina las veces que sea necesarario, explorando una nueva forma de fabricación.

Modelo

Antes de partir con la creación de nuestro molde, debemos crear un modelo 3D, para poder “sacarle copia”. Para ello, en este caso realizaremos un Sketch de nuestra figura y luego realizaremos un revolve, con la configuración que se puede apreciar en la imagen.

Lo que genera un resultado como el de la imagen

Terminando el modelo daremos paso a la creación del molde.

Molde

Para poder crear el molde debemos generar un cuerpo al rededor de nuestro modelo, que contega la forma “en negativo” de nuestro modelo. Para esto, generemos un sketch en la zona inferior de nuestro modelo.

Luego hacemos un extrusión, que cubra contenga totalmente el modelo. Es importante que se seleccione la opción de crear un nuevo cuerpo, para que luego podamos hacer la unión.

Ahora para poder trabajar de mejor manera, bajaremos la opacidad del molde, y pintaremos nuestro modelo.

Para finalmente crear nuestro molde, con el comando de unión debemos realizar un corte entre ambos cuerpos, pero mantiendolos, lo que dará como resultado el molde con el modelo 3D incorporado.

Desmolde

Como podemos notar, el molde coincide a la perfección con nuestro modelo, lo que puede generar inconvenientes a la hora de poder realizar el desmolde. Para evitar lo anterior, crearemos un ángulo de desmolde. Para ello, realizamos un sketch proyectado sobre nuestro modelo, el que usaremos como base para dar con el ángulo.

Luego realizaremos un revolve, pero con la diferencia de que esta vez cortaremos los objetos, para poder generar este ángulo de seguridad a la hora de usar nuestro molde.

El resultado, como podemos apreciar, es una pequeña diferecia entre el molde y el modelo, que permitira un ángulo de salida de nuestro objeto a la hora del desmolde
